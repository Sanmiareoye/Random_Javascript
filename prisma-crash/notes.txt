Prisma commands

prisma generator - generates client code to use in your code

npm install prisma @prisma/client - installs prims and its client
npx prisma init - initialises our prisma schema

npx prisma migrate dev --name init <- apply schema changes to db and give a name to it.
npx prisma generate <- generate prisma client and access it in your code.

npx prisma db seed <- seed the db
